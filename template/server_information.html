安裝 chrome

  • Download stable version for chrome
wget -c https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
  • Install
sudo dpkg -i google-chrome-stable_current_amd64.deb
  • Check the verion
google-chrome --verion
  • Download the corresponding chrome version for crawler from official website
https://chromedriver.chromium.org/

安裝 cuda & cupy

  1. 安裝nvidia驅動
  • Add third party library for nvidia
sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt update
sudo apt upgrade
  • install driver tool
sudo apt install ubuntu-drivers-common
  • check the version of drivers
ubuntu-drivers devices
  • install the recommended version
sudo apt install nvidia-driver-XXX
  1. 安裝cuda
  • Download cuda from official website
https://developer.nvidia.com/cuda-toolkit-archive
  • Reboot
  • Check it is installed or not
nvidia-smi
  1. 安裝cupy
  • Notice: it takes a 20~30min
sudo pip3 install cupy
  1. You can start to use GPU !
